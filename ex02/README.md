# 第2回
## 超高機能電卓(ex02/calc.py)
###　追加機能
- 四則演算:＋以外の四則演算をする
- クリア:entryに入力されている数字、数式の末を1文字deleteする。
- オールクリア:entryに入力されている数字、数式の文字列全体をdeleteする。
- %:入力を％表記として、％表記ではない数字を表示する
- 表示の改善:四則演算以外に追加したボタンの位置を「btn.grid(row=r, column=c)
」を工夫することで綺麗な表示状態にする。
