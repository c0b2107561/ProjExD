# 第四回
## 逃げろこうかとん
### ゲーム概要
- ex04/dodge_bomb.pyを実行すると、1600*900のスクリーンに推す限が描写され、こうかとんを移動させ飛び回る爆弾から逃げるゲーム
- こうかとんが爆弾と接触するとゲームオーバーで終了する
### 操作方法
- 矢印キーでこうかとんを上下左右に移動する

### 追加機能
- 爆弾の追加/速度変更
- ゲームオーバー画面の追加
- ゲームオーバー画面から「shiftキー」を押してゲーム画面へ戻る
